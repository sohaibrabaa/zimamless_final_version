# Zimmamless ML service — pinned, and deliberately all pure-wheel installs.
#
# Every dependency here installs from PyPI with no native toolchain and no
# separately-installed binary. That is a hard constraint, not a preference:
# the OCR requirement (ZM-DOC-005) would otherwise need Tesseract, which on
# Windows means an out-of-band installer and a PATH entry that CI, the demo
# machine, and the deployment target would each have to be told about
# separately. RapidOCR ships its ONNX models inside the wheel, so
# `pip install -r requirements.txt` is the whole setup on every platform.
